Getting Started
Welcome to GptGO! Follow the steps below to get started:
  • Click on in the top right corner of Chrome and choose GptGO from the list
  • Log into your ChatGPT account. If there's no log in page, you're all set!
  • Once you have logged in, you can close the ChatGPT tab and start using GptGO.
Still showing log in page even after logging in?
Try using your personal API key

Warning: this feature is costs money. However, OpenAI grants most new users 3 months free!
  • To see if you have free credits, go here and make sure you see "Free trial Usage".
  • Head over to API Keys to create and copy your API key
  • Congrats! you will now search with your API key instead of using the browser version. If you ever need to change the key you're using, just re-paste it into the box above.
How to Use
The quickest and easiest way to use GptGO is Quick Search:
  • Highlight any section of text with your cursor
  • Right click and select "Search + get response as notification"
  • Within a few seconds, you should receive a notification with ChatGPT's response
Give it a try on the prompt below:
How many moons does Jupiter have?
Tip:
If this doesn't work, your notifications are probably disabled. See the section below to fix.
To read longer responses and search custom prompts, use the GptGO popup:
  • Highlight any section of text with your cursor
  • Open the GptGO popup, the text you highlighted will already be in the prompt
  • If needed, edit your prompt, then click Search or hit ENTER
Give it a try on the prompt below:
Write a paragraph about AI.
Tip 1:
Refresh any pages you had open prior to downloading GptGO
Tip 2:
GptGO can't pick up highlighted text on embedded content such as PDFs and Google Docs. In this case, just copy and paste your text into the prompt section

+
  • Refresh the pages that were open prior to GptGO being installed
  • In some forms of text such as PDF's, Google Docs, and other embedded content, GptGO is unable to retrieve the text you have highlighted. In this case, please copy and paste your desired prompt into the popup
